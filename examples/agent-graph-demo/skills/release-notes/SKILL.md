# Release Notes

Use this skill to summarize a commit range into release notes. Drives the
generic-platform fixture: a root-level `skills/<name>/SKILL.md` is the
fallback for tools that have no platform-specific skill directory.

See @docs/architecture/principles.md for tone and audience guidance.
